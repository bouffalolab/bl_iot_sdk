9.51.5 WebAssembly Module Layout

as will only produce ELF output, not a valid WebAssembly module. It is possible to make as produce output in a single ELF section which becomes a valid WebAssembly module, but a linker script to do so may be preferrable, as it doesn't require running the entire module through the assembler at once.
